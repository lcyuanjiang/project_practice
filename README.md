# project_practice

整个系统开发分成Web端+微信公众号/微信小程序+服务器端（数据库+机器人模型）
1、Web端和微信方面+服务器搭建，我这边搞定
2、你需要完成机器人的训练（Seq2Seq/LSTM/采用注意力机制等），能够完成对话流程（用户问句输入进来，要进行分词，然后意图识别，然后查找知识库/问答库，找到问句，返回问句对应的回答）
3、知识库/问答库可以采用文件或者数据库等任意形式。
